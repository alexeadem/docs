What Are QBO Units?

QBO Units are a universal pricing unit used by QBO to represent consumption of compute resources such as CPU, GPU, RAM, disk, and network. Each QBO Unit is pegged to 1.00 USD as a baseline for pricing transparency and simplicity.

Think of QBO Units as a “currency of compute” — allowing any resource or environment (Cloud or On-Prem) to be priced, billed, and compared using the same unit. This abstraction enables consistent, environment-agnostic cost modeling and lays the groundwork for future blockchain integration or usage-based billing tokens.

Step-by-Step Instructions

  1. Customer Inputs
    • Email: Enter customer’s email.
    • Environment: Choose between Cloud (1.0x) or On-Prem (0.7x). This multiplier affects QBO Unit rates based on deployment location.
    • Discount: Use the −/+ buttons to apply a discount (e.g. 10%, 30%, etc).
      Note: This discount only affects the final USD total. The QBO Unit rates and quantities remain unchanged. This ensures consistent compute accounting, while letting you apply commercial incentives or special pricing per customer.
  2. Resource Configuration
    • CPU Cores: Input number of cores.
    • GPU Models:
      • Select GPU model.
      • Adjust quantity.
      • Click “+” to add another GPU type.
      • Jetson models will override CPU cores automatically.
    • RAM / Disk / Network:
      • Only enabled for Cloud.
      • Values reset to 0 for On-Prem.
  3. Pricing Display
    • QBO Unit Rate: Shown for each resource, adjusted by the environment multiplier.
    • Hourly/Monthly Units: Based on 720 hours per month.
    • USD Table: Shows:
      • 1 QBO Unit = $1.00 base (adjusted by discount for final USD)
      • Total price in USD after discount is applied to the QBO Unit total
  4. Minimum Spend Warning
    • For On-Prem, a red warning appears next to the “Total” row if usage is under 50,000 QBO Units.
  5. Export CSV
    • Click Export CSV to download a quote summary file for the customer.
