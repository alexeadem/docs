What Are QBO Units?

QBO Units are a universal pricing unit used by QBO to represent consumption of compute resources such as CPU, GPU, RAM, disk, and network. Each QBO Unit is pegged to 1.00 USD as a baseline for pricing transparency and simplicity.

Think of QBO Units as a “currency of compute” — allowing any resource or environment (Cloud or On-Prem) to be priced, billed, and compared using the same unit. This abstraction enables consistent, environment-agnostic cost modeling and lays the groundwork for future blockchain integration or usage-based billing tokens.

Step-by-Step Instructions

  1. Customer Inputs
    • Email: Enter the customer’s email address.
    • Environment: Choose between Cloud (1.0x) or On-Prem (0.7x). This multiplier adjusts QBO Unit rates based on where the workload is deployed.
    • Discount: Use the −/+ buttons to apply a percentage discount.
      Note: This discount only affects the final USD total. The QBO Unit rates and quantities remain fixed. This allows sales teams to apply incentives or special pricing without changing the underlying compute valuation.
  2. Resource Configuration
    • CPU Cores: Enter the desired number of CPU cores.
    • GPU Models:
      • Select a GPU model from the dropdown.
      • Adjust the quantity.
      • Click “+” to add another GPU type.
      • If a Jetson model is selected, the CPU field will auto-populate with the corresponding number of cores and become read-only.
    • RAM / Disk / Network:
      • These inputs are only enabled for Cloud environments.
      • When On-Prem is selected, RAM, Disk, and Network values are automatically reset to 0 and disabled. These fields are not currently considered in On-Prem pricing.
  3. Pricing Display
    • QBO Unit Rate: Displayed per resource and adjusted by the environment multiplier.
    • Hourly/Monthly Units: Calculated based on 720 hours per month.
    • USD Table: Shows:
      • 1 QBO Unit = $1.00 base (adjusted by any discount for the USD total)
      • The final USD amount is the total QBO Units multiplied by the discount-adjusted rate
  4. Minimum Spend Warning
    • If On-Prem is selected and the total is under 50,000 QBO Units, a red warning appears next to the “Total” row to indicate the shortfall.
  5. Export CSV
    • Click Export CSV to generate a downloadable quote summary file for the customer.
